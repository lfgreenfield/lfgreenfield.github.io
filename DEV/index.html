Service Locator Map

Map in the field of Nonprofit GIS. Assists local Lexington non-profit daily. Step by Step helps young mothers, ages 14-24, get connected with services, such as education, housing, financial assistance, food, etc. This map is a tool for Step by Step personnel as well as young mothers to use to locate services near to them; available on Step by Step's website. Users input their address and can filter services.

  • See the map: Service Locator Map
  • See the code hosted on Github

Vacant Properties & Spatial Reach of Landlords

In collaboration with the data-driven social justice group, Lexington Housing Studies, this map draws lines between vacant properties in Northeast Lexington and the owners. A negatively pathologized, under-resourced area in central Lexington, this map is a critical analysis of whose 'fault' vacancy and blight really are. These lines show intense concentrated private ownership and mis-maintenance.

  • See the map: Vacant Properties & Spatial Reach of Landlords
  • See the code for this map hosted on Github

Data Representation: Unemployment in Kentucky: 2001 – 2013

This is an interactive, choropleth map showing unemployment rates in Kentucky for the years 2001-2013. The map uses an HTML range slider to sequence through the years and update the map.

  • See the map: Unemployment in Kentucky: 2001 – 2013
  • See the code for this map hosted on Github

About

Hi! My name is Laura. I'm a senior in Geography at the University of Kentucky. My freshman year, I read that there is a social activism aspect to mapping (Crampton 2010) and I was hooked. I've pursued the addition of mapping into my toolbox and have found how they can be applied in any field I'm interested in: urban geography, agriculture, web development.

Maps are powerful tools, especially on the web-based, interactive platform and I want my work to celebrate and support this. My work so far falls under Nonprofit/Community GIS, where I see mapping as information-sharing and putting tool into real-life user's hands and capabilities.The dream is to one day, manage a farm and make maps.
