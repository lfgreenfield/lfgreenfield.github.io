Web Mapping at the University of Kentucky

Power Plants Across the United States

This is an interactive, proportional-symbol map showing natural gas, hydraulic, and nuclear power plants across the U.S. Pop-ups on each power plant provide site-specific data, including name of plant and how much power is produced.

  • See the map: Power Plants Across the United States
  • See the code for this map hosted on Github

Vacant Properties in Kentucky

This is an interactive choropleth map displaying vacant property data for Kentucky. The map uses an HTML drop-down menu to sequence through datasets related to vacant properties in Kentucky.

  • See the map: Vacant Properties in Kentucky
  • See the code for this map hosted on Github

Unemployment in Kentucky: 2001 – 2013

This is an interactive, choropleth map showing unemployment rates in Kentucky for the years 2001-2013. The map uses an HTML range slider to sequence through the years and update the map.

  • See the map: Unemployment in Kentucky: 2001 – 2013
  • See the code for this map hosted on Github

Service Map: Step-By-Step in Lexington, KY

This map, in its development phase, is the start of a project for Step By Step, a community organization in Lexington focused on connecting young mothers (ages 14-24) to services. The map is part of a service mapping movement which aims to provide geospatial services based on the needs of a community organization.

Final functioning map features will include an address search box, info panel displaying attributes for each service, and an ability to filter services according to service category.

  • See the map: Service Map: Step-By-Step
